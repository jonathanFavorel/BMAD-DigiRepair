## Technique Execution Results

### Technique 1 : Alien Anthropologist

**Focus :** Révéler les absurdités de l'industrie de la réparation

**Idées générées :**

**[UX #1]** : Le Formulaire Fantôme
_Concept_ : Les sites de réparation demandent de remplir un formulaire puis silence radio. Le client attend passivement qu'on le rappelle.
_Nouveauté_ : Le standard est un parcours passif/anxiogène - opportunité de réponse immédiate.

**[UX #2]** : Le Prix Mystère
_Concept_ : Presque aucun site n'affiche de prix clairs. "Demandez un devis" partout. La transparence tarifaire est quasi-inexistante alors que c'est le critère n°1.
_Nouveauté_ : La transparence est un différenciateur massif inexploité.

**[UX #3]** : L'Appareil Otage
_Concept_ : Le client dépose son appareil et ne sait plus rien. Pas de suivi, pas de notification. Le tracking existe pour les colis mais pas pour les réparations.
_Nouveauté_ : Le suivi temps réel est attendu partout sauf dans la réparation.

**[SEO #4]** : Le Village Invisible
_Concept_ : Les réparateurs locaux sont invisibles sur Google. Les résultats montrent Paris, Lille, des annuaires. Aucune page dédiée par ville.
_Nouveauté_ : Opportunité SEO locale énorme et inexploitée.

**[Confiance #5]** : Le Diplôme Fantôme
_Concept_ : Pas de certifications visibles, pas de photos d'atelier, pas de vidéos de microsoudure. Confiance aveugle demandée.
_Nouveauté_ : La microsoudure est un vrai métier de précision mais personne ne le montre.

**[Conversion #6]** : Le Client Fantôme du Prix
_Concept_ : Client demande prix → réponse → silence. "80€" seul ne dit rien. "80€, réparé en 30min, garanti 6 mois, pièce certifiée" raconte tout.
_Nouveauté_ : Transformer l'annonce prix en proposition de valeur complète.

**[Conversion #7]** : La Peur du Supplément Caché
_Concept_ : "Y'a pas de frais en plus ?" = question de confiance, pas d'argent. "Prix FINAL, zéro surprise, garanti par écrit" tue l'objection n°1.
_Nouveauté_ : La transparence totale comme argument marketing massif.

**[Process #8]** : Le Devis-Friction
_Concept_ : Obligation légale de devis signé avant réparation. Et si ce processus obligatoire devenait une force ? Devis digital signable en ligne AVANT de se déplacer.
_Nouveauté_ : Transformer la contrainte légale en avantage concurrentiel.

**[Conversion #9]** : Le SMS de la Mort
_Concept_ : Prix par SMS/email → 0% de retour. Le SMS est un canal froid, sans contexte. "80€" dans un SMS = le client compare 5 réparateurs et prend le moins cher.
_Nouveauté_ : Le canal de réponse est LE problème n°1, pas le prix.

**[Process #10]** : L'Artisan Figma
_Concept_ : Devis créés sur Figma + 7 sous-dossiers manuels par client = 1-2h d'admin par client. Le site doit REMPLACER tout ce workflow.
_Nouveauté_ : Le site n'est pas qu'une vitrine - c'est un outil de productivité.

**[Process #11]** : Le Dossier-Cathédrale
_Concept_ : Structure DOS-YYYY-XXXX avec 7 sous-dossiers (admin, docs client, photos dépôt, photos après, échanges, tech, achats) = blueprint parfait pour un back-office digital.
_Nouveauté_ : Transformer la rigueur cachée en expérience client visible.

**[UX #12]** : Le Parcours Fantôme Complet
_Concept_ : Google → site (peut-être) → WhatsApp/appel/SMS/email → prix → FANTÔME. 4 canaux, aucun ne retient. Aucun "crochet" d'engagement avant le prix.
_Nouveauté_ : Capturer l'engagement AVANT la question du prix.

**[Conversion #13]** : Le Prix Nu qui Tue
_Concept_ : "Bonjour, pour votre iPhone X l'écran est à 80€". Le client reçoit ce MÊME message de 5 réparateurs. Zéro différenciation.
_Nouveauté_ : La réponse prix est le moment de vente le plus crucial, traité comme un message automatique.

**[Conversion #14]** : Le Diagnostic Aveugle
_Concept_ : Client décrit en mots, pas de photo. Prix donné sur description orale. Parfois pire que prévu à l'arrivée = mauvaise surprise.
_Nouveauté_ : Diagnostic visuel AVANT le devis change tout pour les deux parties.

**[Productivité #15]** : Le Gouffre des 2 Heures
_Concept_ : 1-2h d'admin par client × 50€/h = 50-100€ de valeur perdue par client. 15 clients/semaine = 750-1500€/semaine de manque à gagner en admin non facturée.
_Nouveauté_ : Le ROI du site = suppression de l'admin + acquisition client.

**[Conversion #16]** : La Réponse qui Convertit vs La Réponse qui Tue
_Concept_ : Au lieu d'un SMS "80€", envoyer un lien vers une page devis personnalisée : prix TTC final, pièce certifiée, garantie, durée, photo upload, signature en ligne, créneau, avis clients.
_Nouveauté_ : Mini-page de vente personnalisée auto-générée au lieu d'un SMS froid.
